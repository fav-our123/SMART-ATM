# Smartsight Accessible ATM

## Overview
Smartsight is an accessible ATM web application designed for visually impaired users. It integrates **audio guidance**, **biometric verification**, and **keyboard-friendly navigation** to provide a fully guided experience. The application is built with **Python (Flask)** and uses **Speech Synthesis (TTS)** for directions.

## Features
- **Step-by-step audio guidance** for all ATM operations.
- **Biometric verification** (simulated) for enhanced security.
- **Balance inquiry**, **cash withdrawal**, and **exit** functionality.
- **Error handling** announced via audio.
- Compatible with **screen readers** (e.g., NVDA) and Braille hardware.

## Tech Stack
- Python 3.x
- Flask
- HTML, CSS, JavaScript
- Browser Text-to-Speech (TTS) APIs
- Optional: Biometric camera access via WebRTC
